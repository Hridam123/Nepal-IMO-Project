# **Longitudinal Analysis of Nepals's Performance and Selection System in the International Mathematics Olympiad**

## **Overview**

This project investigates Nepal's journey in the International Mathematics Olympiad with the help of custom-built data visualization. By utlizing Python(Pandas Library) for data synthesis and Matplotlib for multi-dimensional visualization, this study evaluates Nepal's Mathematics Olympiad Circuit. 
The results of this project led to the publication of 15 page report, which conducts a thorough analysis of Nepal's Math Olympiad journey.

## **Project Objectives**

- Multi-Dimensional Performance Analysis: To develop Python sctipt which calculates the statistical markers such as Mean and Standard deviation to investigate a candidate's performance trends.

- Systematic Reliability Analysis: To form visualizations such as scatter plots and horizantal-line subplots using matplotlib's pyplot to identify the relationship between internal tests(PTST/TST) and international results.
  
- Modelling the Growth Sustainability: To merge nine year of historical data into a combined DataFrame to evaluate not just performance growth but the type of growth in order to forecast growth sustainability.
  
- Computational Reproducibility: To develop a modular Python codebase such that the future data can be easily embedded ensuring that the study evolves with every data and serves as a living diagnostic tool.

## **Key Features and Methodology**

To make sure that the statistics of the project were sound, the following workflow was implemented:

- Automated data Processing: Built a Python workflow such that it cleans and merges mass individual datasets in order to filter out relevant data.
  
- Standarized Scaling: Used the normalization logic in order to convert varied test formats into the standard 42-point(marks) international format to ensure accurate cross-test comparision.
  
- Difficulty-centered Analysis: Engineered a Python workflow such that different questions being analyzed were separated on the basis of their difficulty in order to evaluate difficulty wise performance.

- Visual Representation: Developed series of statistical plots( Bargraphs, Subplots, Scatter plots and line graphs) in order to analyze performance data by visualization.
